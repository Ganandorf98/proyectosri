# ¡Bienvenidos al Lejano Oeste!

Este portal contiene las reglas fundamentales para jugar a **BANG!**, el juego de cartas de tiroteos, roles ocultos y forajidos.

### Resumen rápido
* **Jugadores:** 4 a 7 jugadores.
* **Objetivo:** Depende de tu rol secreto.
* **Preparación:** Cada jugador recibe un personaje con una habilidad especial y una cantidad de balas (puntos de vida).
