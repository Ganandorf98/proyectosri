# Roles y Objetivos

En el Salvaje Oeste, nadie sabe quién es quién (excepto el Sheriff).

| Rol | Objetivo | Identidad |
| :--- | :--- | :--- |
| **Sheriff** | Eliminar a Forajidos y al Renegado. | Visible |
| **Alguacil** | Proteger al Sheriff a toda costa. | Oculta |
| **Forajido** | Eliminar al Sheriff. | Oculta |
| **Renegado** | Ser el último superviviente. | Oculta |

> **Nota:** Si el Sheriff elimina a un Alguacil, ¡pierde todas sus cartas como penalización!
