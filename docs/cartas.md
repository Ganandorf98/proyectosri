# Guía de Cartas

Existen dos tipos de cartas principales. Aprende a usarlas para sobrevivir.

???+ abstract "Cartas de Borde Marrón (Jugar y Descartar)"
    Estas cartas tienen efecto inmediato y luego van a la pila de descartes.

    | Carta | Imagen | Cantidad | Efecto |
    | :--- | :---: | :---: | :--- |
    | **¡BANG!** | ![Bang](img/bang.png){ width="60" } | 25 | Disparo básico a un jugador dentro de tu distancia. |
    | **¡FALLASTE!** | ![Missed](img/missed.png){ width="60" } | 12 | Cancela un disparo de *¡BANG!*. |
    | **CERVEZA** | ![Beer](img/beer.png){ width="60" } | 6 | Recuperas 1 vida. No funciona si solo quedan 2 jugadores. |
    | **CAT BALOU** | ![Cat Balou](img/catbalou.png){ width="60" } | 4 | Obliga a un jugador a descartar una carta (mano o mesa). |
    | **PÁNICO** | ![Panic](img/panic.png){ width="60" } | 4 | Robas una carta a un jugador a distancia 1 (mano o mesa). |
    | **ALMACÉN** | ![Store](img/generalstore.png){ width="60" } | 2 | Se revelan tantas cartas como jugadores vivos; cada uno elige una. |
    | **INDIOS** | ![Indians](img/indians.png){ width="60" } | 2 | Todos los demás deben descartar un *¡BANG!* o perder una vida. |
    | **DUELO** | ![Duel](img/duel.png){ width="60" } | 3 | Desafías a otro: tiráis *¡BANG!* por turnos hasta que uno no pueda y pierda vida. |
    | **AMETRALLADORA** | ![Gatling](img/gatling.png){ width="60" } | 1 | Dispara un *¡BANG!* a todos los demás jugadores (sin importar distancia). |
    | **SALÓN** | ![Saloon](img/saloon.png){ width="60" } | 1 | Cura 1 vida a **todos** los jugadores (incluyéndote). |
    | **DILIGENCIA** | ![Stage](img/stagecoach.png){ width="60" } | 2 | Robas 2 cartas del mazo. |
    | **WELLS FARGO** | ![Wells](img/wellsfargo.png){ width="60" } | 1 | Robas 3 cartas del mazo. |

??? abstract "Cartas de Borde Azul (Equipo y Armas)"
    Se colocan frente a ti y su efecto es permanente hasta que te las quiten.

    | Carta | Imagen | Cantidad | Efecto |
    | :--- | :---: | :---: | :--- |
    | **BARRIL** | ![Barrel](img/barrel.png){ width="60" } | 2 | "Desenfundas" cuando te disparan. Si sale Corazón, esquivas. |
    | **CÁRCEL** | ![Jail](img/jail.png){ width="60" } | 3 | Inhabilita a un jugador. Debe desenfundar: si no sale Corazón, pierde el turno. |
    | **DINAMITA** | ![Dynamite](img/dynamite.png){ width="60" } | 1 | Da vueltas a la mesa. Si "desenfundas" Picas entre 2 y 9, te explota (3 daños). |
    | **MUSTANG** | ![Mustang](img/mustang.png){ width="60" } | 2 | Los demás te ven a distancia +1 (te alejas). |
    | **MIRA** | ![Scope](img/scope.png){ width="60" } | 1 | Ves a los demás a distancia -1 (te acercas). |
    | **VOLCANIC** | ![Volcanic](img/volcanic.png){ width="60" } | 2 | Arma. Alcance 1. Permite jugar cuantos *¡BANG!* quieras por turno. |
    | **SCHOFIELD** | ![Schofield](img/schofield.png){ width="60" } | 3 | Arma. Alcance 2. |
    | **REMINGTON** | ![Remington](img/remington.png){ width="60" } | 1 | Arma. Alcance 3. |
    | **CARABINA** | ![Carabine](img/carabine.png){ width="60" } | 1 | Arma. Alcance 4. |
    | **WINCHESTER** | ![Winchester](img/winchester.png){ width="60" } | 1 | Arma. Alcance 5. |
