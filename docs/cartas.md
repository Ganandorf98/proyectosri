# Guía de Cartas

El mazo de BANG! se divide principalmente en dos tipos de cartas según el color de su borde. ¡Aprende a distinguirlas para ganar el duelo!

???+ abstract "Cartas de Borde Marrón (Efecto inmediato)"
    Estas cartas se juegan, se aplica su efecto y se descartan automáticamente.

    | Carta | Imagen | Efecto |
    | :--- | :---: | :--- |
    | **¡BANG!** | ![Bang](img/bang.png){ width="80" } | El pilar del juego. Sirve para disparar a un jugador a tu alcance. |
    | **¡FALLASTE!** | ![Fallaste](img/missed.png){ width="80" } | La única forma de evitar perder un punto de vida tras un disparo. |
    | **CERVEZA** | ![Cerveza](img/beer.png){ width="80" } | Recuperas un punto de vida. Se puede usar fuera de tu turno si recibes un disparo letal. |
    | **DILIGENCIA** | ![Diligencia](img/stagecoach.png){ width="80" } | Robas 2 cartas del mazo inmediatamente. |
    | **WELLS FARGO** | ![Wells Fargo](img/wellsfargo.png){ width="80" } | Robas 3 cartas del mazo inmediatamente. |

??? abstract "Cartas de Borde Azul (Equipo y Objetos)"
    Estas cartas se colocan frente a ti. Su efecto permanece activo hasta que alguien las robe, las destruya o tú mueras.

    | Carta | Imagen | Efecto |
    | :--- | :---: | :--- |
    | **MUSTANG** | ![Mustang](img/mustang.png){ width="80" } | Aumenta en 1 la distancia a la que te ven los demás. Eres más difícil de alcanzar. |
    | **BARRIL** | ![Barril](img/barrel.png){ width="80" } | Te permite "desenfundar" (sacar carta del mazo) cuando te disparan. Si sale Corazón, esquivas el tiro. |
    | **MIRA** | ![Mira](img/scope.png){ width="80" } | Reduce en 1 la distancia a la que ves a los demás. ¡Todos están más cerca! |
    | **DINAMITA** | ![Dinamita](img/dynamite.png){ width="80" } | Se pasa de jugador en jugador. Si explota (sacas de 2 a 9 de Picas), pierdes 3 vidas. |
    | **ARMAS** | ![Armas](img/weapons.png){ width="80" } | Como la *Volcanic* o la *Winchester*. Modifican tu distancia de alcance o límites de disparo. |
